# 交易教练 — 产品需求文档 (PRD)

## 1. 产品概述

**产品名称**：交易教练（TradeCoach）
**产品定位**：基于 AI 的股票交易复盘微信小程序，帮助交易者建立系统化的复盘习惯、识别认知偏差、提升知行合一能力。
**目标用户**：A 股个人交易者，有一定交易经验但缺乏系统复盘习惯。
**核心价值**：每日 5 分钟复盘 → AI 深度诊断 → 行为模式追踪 → 持续改进。

## 2. 产品架构

### 2.1 页面结构
```
TabBar（4 页）:
├── 复盘（index）     — 核心入口，交易记录表单
├── 数据（dashboard） — 数据仪表盘，可视化指标
├── 自选（stocks）    — 自选股管理，实时行情
└── 我的（profile）   — 设置、数据管理、关于

非 TabBar（4 页）:
├── AI教练（coach）       — AI 分析结果展示
├── 复盘详情（history）   — 单条复盘查看/编辑
├── 周期复盘（period）    — 周/月维度 AI 总结
└── 对话历史（conversation） — AI 对话记录
```

### 2.2 核心数据流
```
用户填写表单 → 自动保存草稿 → 提交 → XML 序列化
    → AI 教练分析（DeepSeek API）→ 流式展示
    → 保存复盘记录 + 对话记录 + 行为标签
    → 数据仪表盘实时计算
```

## 5. 技术约束

- 微信小程序原生框架，无构建工具
- 本地 Storage 存储，无后端
- DeepSeek API 直连（客户端暴露 Key，生产需代理）
- 东方财富公开行情 API
- 基础库版本 3.15.2

## 6. 质量标准

### 6.1 UI 质量
- 所有页面统一设计语言
