## 💡 这是什么

交易教练是一款帮助个人交易者建立**系统化复盘习惯**的微信小程序。

它解决的核心问题是：大多数交易者知道自己应该复盘，但要么不知道怎么复盘，要么坚持不下来。交易教练用 **AI Agent** 做你的私人教练——你只需要如实记录今天做了什么，AI 教练"严正"会自动调用工具获取你的历史数据和市场行情，然后结合这些数据进行深度分析，最后还会自我反思检查有没有遗漏。

**不是"调一次 API 拿结果"，而是一个真正的 AI Agent。**

## 🤖 AI Agent 架构（核心亮点）

这个项目最核心的技术亮点是 **AI Agent 引擎**，实现了两个 Agent 领域的关键模式：

### Harness（驾驭）

AI 不是直接回答问题，而是**先调用工具获取数据**，再基于数据回答。

```
用户提交复盘 → AI 分析需要哪些数据 → 调用工具 → 基于数据生成分析
```

**5 个注册工具：**

| 工具 | 职责 | 返回数据 |
|------|------|---------|
| `get_review_history` | 获取最近 N 条复盘 | 买卖操作、标签、自评 |
| `get_pending_questions` | 获取未回答的追问 | 问题列表 |
| `get_market_data` | 获取实时 A 股行情 | 指数、涨跌停、板块 |
| `get_trading_stats` | 获取统计数据 | 执行率、连续天数、标签分布 |
| `get_tag_trend` | 获取标签趋势 | 某个标签的出现频率变化 |

工具设计理念和 **OpenAI Function Calling** 一致：每个工具有 `name`、`description`、`parameters`、`execute`。

### Loop（循环）

AI 生成分析后，**自我反思检查有没有遗漏**，如有则补充。

```
生成分析 → 自我审查 → "有没有遗漏？" → 有则补充 → 最终输出
```

### 三阶段流程

```text
Phase 1: 工具调用（Harness）
  AI 分析用户复盘 → 决定调用哪些工具 → 执行工具获取数据

Phase 2: 分析生成
  AI 基于工具数据生成深度分析（引用具体数字，不是空话）

Phase 3: 自我反思（Loop）
  AI 检查自己的输出 → "是否遗漏了用户提到的某个操作？" → 有则补充
```

**UI 上可以看到这个过程：** 教练页面会实时显示"收集数据 → 生成分析 → 自我检查"三个阶段的进度。

## ✨ 产品特性

### 📝 结构化复盘

不是让你面对一张白纸发呆。系统引导你逐步记录：

| 步骤 | 内容 | 为什么重要 |
|------|------|-----------|
| 市场分析 | 大盘走势、题材轮动 | 建立全局视角 |
| 知行合一 | 买入/卖出/未执行计划 | 核心：你说了什么 vs 你做了什么 |
| 自我评价 | 打分 + 反思 | 培养元认知能力 |
| 明日计划 | If-Then 规则 | 把反思转化为行动 |

填写过程中**自动保存草稿**，不怕写到一半退出。每日打开时还会自动拉取市场数据预填，减少重复输入。

### 🤖 AI 教练"严正"

不是那种"建议你注意控制风险"的废话 AI。严正教练会：

- **工具驱动**：自动调用工具获取你的历史数据和市场行情，基于真实数据而非空谈
- **纵向对比**：通过 `get_review_history` 工具拉取历史复盘，发现进步和老毛病
- **追问未执行计划**：通过 `get_pending_questions` 工具获取未回答的追问，连续追踪
- **识别认知偏差**：锚定效应、损失厌恶、过度自信……用你今天的操作当证据
- **自我反思**：生成分析后自动检查遗漏，确保不放过任何问题
- **记忆闭环**：上次没回答的追问，下次复盘会自动提醒你

### 📊 数据仪表盘

所有复盘数据实时计算，无需后端：

- **核心指标**：复盘天数、交易次数、计划执行率、连续复盘天数
- **执行率趋势**：14 天滚动趋势图，一眼看清执行力变化
- **行为标签云**：高频标签醒目展示，行为模式一目了然
- **交易排行**：最常交易的股票 Top 10
- **市场概览**：上证/深证指数、涨跌停统计、板块排行

### 📈 自选股管理

- 东方财富联想搜索，输入代码或名称即可查找
- 实时行情刷新，涨跌幅一目了然
- 一键添加/移除，自选股列表本地持久化

### 🔄 周期复盘

支持**周复盘**和**月复盘**两种维度，AI 自动聚合分析：

- 包含的每日复盘列表
- 行为标签频次统计
- 计划执行力趋势表
- AI 综合分析报告

### 🛡️ 数据安全

- 所有数据存储在本地 Storage，不上传任何服务器
- 支持 JSON 格式导出/导入，方便备份迁移
- 存储用量实时监控，接近上限时提醒清理

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone git@github.com:XN-289/finance-coach-mini.git
cd finance-coach-mini
```

### 2. 配置 API Key

```bash
cp utils/config.example.js utils/config.js
```

编辑 `utils/config.js`，填入你的 DeepSeek API Key：

```javascript
const AI_API_KEY = 'sk-your-api-key-here'
const AI_API_URL = 'https://api.deepseek.com/chat/completions'
const AI_MODEL = 'deepseek-chat'
```

### 3. 导入微信开发者工具

1. 打开**微信开发者工具**
2. 选择「导入项目」
3. 目录选择项目根目录
4. AppID 填写：`wxd75aa4f8efb2aeb8`
5. 点击「编译」即可运行

> 无构建步骤、无 npm 依赖、无编译配置——直接打开就能跑。

## 🧠 AI 教练设计

### 教练人设

"严正"——严肃直击要害的交易教练。不讲正确的废话，只讲交易者真正需要面对的真相。

### 分析框架（5 步）

1. **行为对比** — 纵向对比近期操作，发现进步和老毛病
2. **未执行计划深挖** — 追问用户说了但没做的事
3. **知行合一审计** — 评估符合计划的比例变化趋势
4. **认知偏差诊断** — 识别过度自信、损失厌恶、锚定效应等行为金融学偏差
5. **市场匹配度** — 判断操作方向是否与大盘走势一致

### 输出结构

```text
对比诊断（2-3 句话）
今日操作审计
认知陷阱
必须回答的问题（2-3 个）
改进清单（3 条可量化行动）
今日金句
__TAGS__:["标签1","标签2","标签3"]
```

### 行为标签池

```text
追涨、杀跌、止损拖延、止盈过早、过度交易、犹豫不决、
锚定效应、损失厌恶、过度自信、确认偏误、近因效应、
羊群效应、计划缺失、逆势操作、仓位失控、盘中冲动、
报复交易、踏空焦虑
```
