# 交易教练 — AI 驱动的股票交易复盘小程序

一款基于 AI 的股票交易复盘微信小程序，帮助交易者建立系统化的复盘习惯，识别行为偏差，提升知行合一的能力。

## 功能特性

### 核心功能

- **每日交易复盘** — 结构化录入买入/卖出/未执行计划，支持草稿自动保存
- **AI 教练分析** — 集成 DeepSeek API，AI 教练"严正"结合历史复盘进行纵向对比分析
- **行为标签追踪** — AI 自动提取行为偏差标签（追涨、杀跌、损失厌恶等 18 种），长期追踪行为模式
- **教练记忆闭环** — AI 提取的追问会注入下一次复盘 prompt，形成连续对话

### 数据分析

- **数据仪表盘** — 核心指标卡片、执行率趋势图、周活跃度柱状图、行为标签云
- **交易统计引擎** — 实时计算 15+ 指标：胜率、执行率、连续复盘天数、月度趋势等
- **周期复盘** — 支持周/月维度的 AI 聚合分析，含计划执行力趋势表

### 行情与自选

- **实时行情** — 集成东方财富 API，自动获取上证/深证指数、涨跌停统计、板块排行
- **市场数据自动填充** — 复盘时自动拉取当日行情数据填入表单
- **自选股管理** — 股票联想搜索、自选列表、批量实时行情刷新

### 工程能力

- **自定义组件库** — stat-card / mini-chart / tag-cloud / progress-bar 四个可复用组件
- **主题系统** — CSS 变量驱动的亮色/暗色模式切换
- **数据管理** — 导出/导入/备份/清理，解决本地 Storage 数据迁移问题
- **流式 AI 输出** — 打字机效果展示 AI 回复

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | 微信小程序原生框架（自定义组件、懒加载） |
| AI | DeepSeek Chat API，Prompt Engineering |
| 行情 | 东方财富公开 API（指数/涨跌停/板块/个股搜索） |

## 快速开始

### 1. 克隆项目

```bash
git clone git@github.com:XN-289/finance-coach-mini.git
```

### 2. 配置 API Key

复制配置模板并填入你的 DeepSeek API Key：

```bash
cp utils/config.example.js utils/config.js
```

编辑 `utils/config.js`：

```javascript
const AI_API_KEY = 'your-deepseek-api-key'
const AI_API_URL = 'https://api.deepseek.com/chat/completions'
const AI_MODEL = 'deepseek-chat'
```

### 3. 导入微信开发者工具

1. 打开**微信开发者工具**
2. 选择「导入项目」
3. 目录选择项目根目录，AppID 填 `wxd75aa4f8efb2aeb8`
4. 编译运行

无构建步骤、无 npm 依赖，直接编译即可。

## 数据流

```
用户填写表单 → 自动保存草稿 → 序列化为 XML
        ↓
    提交到 AI 教练页面
        ↓
    拉取历史5条复盘 + 未回答追问 + 实时行情
        ↓
    调用 DeepSeek API（System Prompt + 用户数据）
        ↓
    流式展示 AI 回复 → 解析行为标签 + 追问
        ↓
    保存复盘记录 + 对话记录 → 清除草稿
        ↓
    仪表盘实时计算统计指标
```

## AI 教练 Prompt 设计

System Prompt 定义了"严正"教练的5步分析框架：

1. **行为对比** — 纵向对比近期操作，发现进步和老毛病
2. **未执行计划深挖** — 追问用户说了但没做的事
3. **知行合一审计** — 评估符合计划的比例变化趋势
4. **认知偏差诊断** — 识别过度自信、损失厌恶、锚定效应等行为金融学偏差
5. **市场匹配度** — 判断用户操作方向是否与大盘走势一致

AI 回复末尾输出结构化标签 `__TAGS__:["标签1","标签2"]`，正则提取后用于统计分析。
